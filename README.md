## **💬 프로젝트 소개**


💡 옷장 속에 어떤 옷이 있는지 한 눈에 파악하고 해당 데이터 기반으로 코디를 추천해주는 서비스


## ✍🏻 기획 배경

> 계절이 바뀔 때마다 옷을 구매한다
옷장 속의 옷의 50%는 입지 않지만, 무슨 옷을 가지고 있는지 모른다
> 
- 자주 안 입는 옷을 활용할 수 있도록 새로운 코디 추천하는 프로젝트
- 옷장 속의 옷을 한 눈에 파악하여 직관적으로 확인해볼 수 있는 프로젝트
